推荐测试路径：

1. 使用商务登录，第一栏目进入“返点流水”，可筛选流水和查看关联客户。
2. 进入“客户邀约”，点击“生成邀请链接”弹窗创建链接。
3. 在左侧“原型操作”点击“打开客户绑定页”，模拟客户进入独立入口并提交资料。
4. 切换到总后台，进入“官方终审”，通过或驳回客户归属审批。
5. 审批通过后查看“我的客户”“渠道结算”“客户渠道归属”“审计日志”。
6. 进入“规则与协议”，查看总后台配置的当前角色规则说明文本。
7. 也可以使用左侧“原型操作”的“跑通主流程”，它会按真实步骤自动代填。

原型说明：

- 仅前端本地状态，不接后端。
- 左侧“原型操作”和“数据状态”只用于演示和测试，不属于正式后台功能。
- 数据保存在浏览器本地，可在“原型操作”中点击“重置演示数据”恢复初始状态。
- 页面保留“现有功能”和“本期优化点”标签，用于区分当前已有入口和本期要补齐的闭环能力。
